called via block explorer or scripts. UI implementation not required:

   - Updates fee collection address
   - Admin only operation

   - Adjusts MEV protection cooldown (20-86400 seconds)
   - Admin only operation

4. updateListingFee(uint256)
   - Changes listing fee (within immutable limit)
   - Admin only operation

5. updateProtocolFee(uint256)
   - Changes protocol fee in basis points (within immutable limit)
   - Admin only operation

6. cleanInactiveUsers(uint256, uint256)
   - Performance optimization for findFillableOrders
   - Admin maintenance function

7. addTokenAddress(address) [Whitelist]
   - Adds new token to whitelist
   - Admin only operation

8. setTokenStatus(address, bool) [Whitelist]
   - Activates/deactivates whitelisted tokens
   - Admin only operation
